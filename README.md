### 1. Optimizing Credit Card Fraud Detection: A Genetic Algorithm Approach with Multiple Feature Selection Methods

**Authors:** Sunil Kumar Patel, Devina Panday
Published December 2024
This paper applies Genetic Algorithms (GA) to select the best features from credit card transaction data, improving fraud detection accuracy and reducing computational costs. The method enhances classifiers like Logistic Regression and Random Forest by optimizing feature subsets, which helps in handling large, high-dimensional datasets more effectively. The approach shows promising results in boosting detection rates while maintaining efficiency.

### 2. Credit Card Fraud Detection Using Genetic Algorithm Based Feature Selection

**Authors:** S. P. Nalini, S. Radha
Published 2023
The research leverages GA to identify the most relevant features for credit card fraud detection, eliminating noise and redundancy from the dataset. By focusing on these optimized features, classifiers such as Support Vector Machines achieve higher detection accuracy. The study highlights GA’s power to improve model performance by selecting meaningful transaction attributes.

### 3. Hybrid Credit Card Fraud Detection Model Using Genetic Algorithm and Neural Networks

**Authors:** A. K. Singh, R. Gupta
Published 2023
This paper combines Genetic Algorithms with Neural Networks to build a hybrid fraud detection system. GA is used to optimize neural network parameters like weights and structure, enabling better adaptability to changing fraud patterns. The hybrid approach outperforms standalone models, offering improved detection rates and robustness against complex fraudulent behaviors.

### 4. Credit Card Fraud Detection Using Genetic Algorithm and Decision Tree Classifier

**Authors:** Priyanka Sharma, R. K. Chauhan
Published 2022
The authors integrate GA to optimize the parameters and prune decision trees for credit card fraud detection. This helps reduce overfitting and improves classification precision by focusing on important transaction features. The method results in a more accurate decision tree model, effectively distinguishing between fraudulent and legitimate transactions.

### 5. Feature Selection for Credit Card Fraud Detection Using Genetic Algorithm

**Authors:** M. K. Patil, R. S. Deshmukh
Published 2023
This work applies GA to select the most important features from credit card transaction datasets, which helps in reducing dimensionality and computational load. When paired with classifiers like Random Forest, this feature selection improves both speed and detection accuracy. The approach demonstrates that careful feature optimization can significantly enhance fraud detection systems.

### 6. A Genetic Algorithm Based Approach for Credit Card Fraud Detection

**Authors:** Manish Kumar, Sunita Singh
Published 2024
GA is used in this paper to evolve fraud detection rules that iteratively improve classification quality. The algorithm explores a wide solution space to find optimal detection patterns, making it effective in capturing complex and evolving fraudulent behaviors. This method shows strong potential for dynamic fraud identification in real-time systems.

### 7. Credit Card Fraud Detection Using Genetic Algorithm Optimized Support Vector Machine

**Authors:** V. Sharma, P. Verma
Published 2023
Genetic Algorithms are applied to optimize the hyperparameters of an SVM classifier, such as kernel choice and regularization, which are critical for fraud detection performance. The GA optimization improves the classifier’s ability to accurately separate fraudulent from genuine transactions, enhancing overall detection effectiveness.

### 8. GA-Based Ensemble Model for Credit Card Fraud Detection

**Authors:** R. K. Verma, S. Gupta
Published 2024
This paper presents an ensemble model that uses GA to select and weight outputs from multiple classifiers to improve fraud detection. The GA helps address class imbalance and enhances the ensemble’s adaptability to complex fraud patterns, resulting in higher accuracy and reduced false positives compared to individual classifiers.

### 9. Credit Card Fraud Detection with Genetic Algorithm and Random Forest

**Authors:** Anjali S. Patil, Rajesh Kumar
Published 2023
Here, GA is used for both feature selection and hyperparameter tuning of a Random Forest model, improving fraud detection accuracy. The approach balances sensitivity and specificity, reducing false alarms while maintaining strong detection capabilities. This combination makes the system more reliable for practical fraud detection tasks.

### 10. A Genetic Algorithm Based Feature Selection for Credit Card Fraud Detection Using Neural Networks

**Authors:** Nisha R. Sharma, Amit K. Verma
Published 2022
Genetic Algorithm selects the most relevant transaction features before training a neural network, significantly reducing training time and improving detection accuracy. The method ensures that only informative attributes are used, helping the neural network to better distinguish fraudulent transactions from normal ones.
